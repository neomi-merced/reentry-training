# Re-Entry & STRE Processing Training

An interactive training module for Nightingale College Registrar staff 
covering Traditional Re-Entry, Short-Term Re-Entry (STRE), and Status 
Update processing workflows.

## Live Training
👉 [Launch Training](https://neomi-merced.github.io/reentry-training/)

## What's Inside

- 3 briefing pages covering Standard Re-Entry, STRE, and Status Updates
- 6 branching scenarios with real-world learner profiles
- Score tracking — earn points for first-try correct answers
- 12-card interactive flashcard study deck
- Quick Reference guide with all key rules and thresholds
- Feedback form embedded at completion

## Topics Covered

- Traditional Re-Entry eligibility and LDA window rules
- TCE modification rules based on LDA window
- Transfer credit labeling — TC vs Taken @ NC
- HESI and MDC exam credit eligibility
- STRE eligibility criteria and the copy-paste TCE rule
- Re-Entry Wizard workflow for status updates

## Built With

- Twine — SugarCube story format
- Custom CSS with light and dark mode support
- JavaScript for score tracking and flashcard navigation
- GitHub Pages for hosting
